General

Information on how to change colors, edit CSS,etc.

Colors

The primary color of this template is #83E8BA. It provides great contrast against the dark grey background of the template. You can change this color by creating a new rule at the end of the CSS file and targeting the text-primary and bg-primary CSS rules. See below: 

                    
                        // At the end of styles.css
                        .text-primary {
                            color: red;
                        }

                        .bg-primary {
                            background-color: red;
                        }

                        // Changing the color of an element
                        .element {
                            color: blue;
                        }

Note, however, that the primary color is also used when styling the elements itself. So to change it, you'll need to search through the CSS file, starting from the section labeled Custom CSS and change the green from there from there. You can also use the Find and Replace function on any text editor.

If you have SCSS, just change the colors inside of the variables.scss file in the /scss folder. 

                    
                        $primary: blue;

Icons

Icons are provided by ionic icons. To change any icon on the page, simply change the name attribute of any ion-icon element. Please note, icons are not animated.

                    
                            // Will show a mail icon
                            <ion-icon name="mail" ></ion-icon>

Get a list of icon names here: Ionicons

Changing Images

Changing Images throughout the document

This is my favourite part. I've made changing images extremely easy for my users. To change an image, simply look for the data-image attribute on any html element and change the url inside of it to the url of your image. No need to go inside a CSS file. 

                
                    <hero data-image="your-image.jpg">

This simply sets a background-image to the element. This makes it easier to scale and use without cutting any piece of the image off.

Changing Video (for video template)

Changing the Video in the Jumbotron

To change the video, simply change the url in the src 

                
                    <video autoplay muted loop class="jumbotron-video">
                        <source src="your-video.mp4" type="video/mp4">
                    </video>
